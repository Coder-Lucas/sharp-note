# SharpNote

SharpNote是一款开源在线笔记应用，使用Markdown格式，并且极其注重个人隐私，所有数据全部位于本机储存。

---

## 立即使用SharpNote

- 网站链接：[https://shnt.vercel.app/](https://shnt.vercel.app/)
- GitHub仓库：[https://github.com/Coder-Lucas/sharp-note.git](https://github.com/Coder-Lucas/sharp-note.git)

---

## 获取源代码

### 克隆仓库

```bash
# 请确保已安装Git
git clone https://github.com/Coder-Lucas/sharp-note.git sharp-note
```

### 安装依赖

```bash
# 进入项目目录
cd "./sharp-note"
# 请确保已安装npm
npm i
```
