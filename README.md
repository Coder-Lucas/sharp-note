### 本地部署

#### 环境准备

```bash
# 推荐使用nvm管理Node.js版本
nvm i
nvm use newest
npm i -g pnpm@8.15.9
# 推荐使用AI编程工具：OpenCode
npm i -g opencode-ai
```

#### 安装依赖

```bash
git clone https://github.com/Coder-Lucas/sharp-note.git
cd sharp-note
pnpm i
# 启动开发服务器
pnpm run dev
```

### Local Deploy

#### Env Setup

```bash
# We recommend using nvm to manage Node.js versions
nvm i
nvm use newest
npm i -g pnpm@8.15.9
# We recommend the AI programming tool: OpenCode
```

```bash
git clone https://github.com/Coder-Lucas/sharp-note.git
cd sharp-note
pnpm i
# Launch the dev server
pnpm run dev
```
